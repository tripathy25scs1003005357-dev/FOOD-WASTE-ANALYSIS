🍽️ Food Waste Analysis Project
📌 Overview

This project focuses on analyzing food waste behavior among individuals using real-world survey data. The goal is to identify patterns, understand key reasons behind food waste, and provide actionable insights to reduce it.

The dataset consists of responses collected from students through a structured survey (Google Forms), followed by data cleaning, analysis, and visualization using Python.

🎯 Objectives
Analyze food waste patterns among students
Identify major causes of food waste
Perform data cleaning and preprocessing
Generate visual insights using graphs and charts
Provide recommendations to reduce food waste
📊 Dataset
Source: Primary data collected via survey
Size: 100+ responses
Format: Excel (.xlsx) / CSV
Features include:
Age
Gender
Food waste type
Frequency of waste
Quantity of waste
Reasons for food waste
⚙️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Excel (for dataset and visualization support)
🔄 Project Workflow
Data Collection (Google Forms)
Data Cleaning (handling missing values, duplicates)
Data Preprocessing
Exploratory Data Analysis (EDA)
Data Visualization (bar graphs, pie charts, line graphs)
Interpretation of results
📈 Key Visualizations
Food Waste Distribution (Bar Chart)
Percentage of Waste Types (Pie Chart)
Waste Frequency Trend (Line Graph)
Age-wise Analysis
Gender-wise Analysis
Reasons for Food Waste
📌 Key Insights
Certain food categories are wasted more frequently
Food waste is influenced by habits and portion sizes
Major reasons include over-serving, taste issues, and poor planning
Awareness and portion control can significantly reduce waste
💡 Conclusion

This project highlights the importance of understanding food consumption behavior. By analyzing patterns and reasons, it becomes easier to implement strategies that minimize food waste and promote sustainable habits.
